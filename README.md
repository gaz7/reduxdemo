# reduxdemo

This is a demo site for trying out React with Redux.
Also incorporates remote calls

## Running locally
1. git clone git@github.com:gaz7/reduxdemo.git
1. cd reduxdemo
1. vagrant up
1. vagrant ssh
1. cd /vagrant
1. npm start
1. Access at http://redux.manteka.org:3000/
